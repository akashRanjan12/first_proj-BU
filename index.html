1. Machine Language: The lowest level of programming language that computers understand directly. It consists of binary code (0s and 1s) and corresponds directly to the machine's hardware instructions.
2. Assembly Language A low-level programming language that uses mnemonic codes to represent machine instructions. It's more human-readable than machine language but still closely tied to the computer's architecture.
3. High-Level Languages These languages are closer to human languages and abstract away the complexities of machine instructions. - Examples: C, C++, Java, Python, Ruby, JavaScript. - Characteristics: Easier to read and write, platform-independent (in most cases), and often come with extensive libraries and frameworks. - Usage: Widely used for developing a variety of software applications, from system software to web applications, mobile apps, and more.
4. Scripting Languages A type of high-level language that is interpreted rather than compiled. - Examples: JavaScript, Python, Ruby. - Characteristics: Often used for automating tasks, web development, and rapid prototyping due to their ease of use and dynamic typing.
5. Domain-Specific Languages (DSLs) Languages designed for a specific domain or problem set. - Examples: SQL (for database queries), HTML/CSS (for web development), MATLAB (for numerical computing). - Characteristics: Tailored to specific tasks or industries, often optimized for particular types of problems.
